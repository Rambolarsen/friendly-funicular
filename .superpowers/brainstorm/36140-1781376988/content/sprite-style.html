What visual direction for the sprites?

All options can work within the existing Phaser setup. Pick the vibe that fits the game.

A — Pixel Art

Classic retro look. Characters drawn pixel-by-pixel using Phaser's graphics API. Fully self-contained, no external files.

free asset packs (kenney.nl)

B — Free Sprite Assets

Real PNG sprites from Kenney.nl (CC0 public domain). Polished look instantly. Requires adding image files to the project.

C — Geometric SVG-style

Bold shapes with personality. SVGs embedded and rasterized by Phaser at load time. Self-contained, more expressive than plain rectangles.
